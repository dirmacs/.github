# DIRMACS organization defaults

This repository holds default community health files and the **organization profile README** for [DIRMACS](https://github.com/dirmacs).

- **`profile/README.md`** — Shown on the [organization landing page](https://github.com/dirmacs).
- **Root files** — Optional org-wide defaults (e.g. CONTRIBUTING, CODE_OF_CONDUCT) for repositories that don't define their own.
